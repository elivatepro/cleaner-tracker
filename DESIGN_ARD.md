## 1. Design Principles

1. **Speed over beauty** â€” Cleaners interact with this app in 10-30 seconds. Every screen must be actionable immediately.
2. **One action per screen** â€” Each screen has one primary action. No clutter, no distractions.
3. **Thumb-friendly** â€” All primary actions within thumb reach on mobile. Large tap targets (min 48x48px).
4. **High contrast** â€” Black and white default. Readable in bright sunlight (outdoor workers).
5. **Status at a glance** â€” Cleaner should know their status (checked in / not checked in) within 1 second of opening the app.
6. **White-label ready** â€” Colors, logo, and company name are configurable without code changes.

## 8. Micro-interactions & Feedback

### 8.1 Check-in Button
- Idle: PRIMARY bg, white text, subtle shadow
- Loading: Spinner replaces text, button disabled
- GPS acquiring: Pulsing location icon with "Getting your location..."
- Success: Button turns ACCENT (green), shows checkmark, then transitions to checked-in state
- Failed (outside geofence): Button shakes briefly, WARNING state appears

### 8.2 Checkout Checklist
- Checkbox tap: Satisfying fill animation (scale 0â†’1 with checkmark)
- All items checked: Subtle confetti or green glow on COMPLETE button

### 8.3 General
- Page transitions: Fade (0.2s ease)
- Cards: Hover lift (translateY -2px) on desktop
- Buttons: Press scale (0.97) on tap
- Toasts: Slide in from top, auto-dismiss after 3s
- Skeleton loading: Pulse animation for content placeholders

## 10. Accessibility Checklist

- [ ] All images have alt text
- [ ] Color is not the only indicator of status (icons + text accompany colors)
- [ ] Focus states visible on all interactive elements (2px PRIMARY outline)
- [ ] Form labels associated with inputs
- [ ] Error messages announced to screen readers (aria-live)
- [ ] Minimum 4.5:1 contrast ratio for body text
- [ ] Minimum 3:1 contrast ratio for large text and UI components
- [ ] Touch targets minimum 48x48px
- [ ] Keyboard navigation works for all admin panel interactions
- [ ] Skip to content link on admin layout
